# Insert Row

Rows can be inserted to the database using the `INSERT` statement in superinsight.

### Insert Single Row
```
INSERT INTO mldb.table_name(column1, column2, …)
VALUES (value1, value2, …);

```


### Insert Single Row
```
INSERT INTO mldb.table_name (column_list)
VALUES
    (value_list_1),
    (value_list_2),
    (value_list_n);

```
